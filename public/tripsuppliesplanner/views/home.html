Trip Supplies Packing List

This website is designed to help generate an appropriate packing list for any vacation you may go on. Normally you have to spend time trying to decide what you need to pack, and almost always forget to pack something important. Now with this, if you forget something once, you can add it to our list, and next time it will remember for you!

NOTE: This site is only in its alpha stages. So if you find any problems or have any questions please contact me.

Working Features:

  • Creating of a Vacation
  • Generating a packing list for specified vacation
  • Create new items to use for vacations
  • Create Personal Items (only for you) to apply to vacations

Upcoming Features:

  • Ability to create family groups
  • Add children (people who can't log in) to family
  • Invite other people to your vacation
  • Create a vacation without signing in
