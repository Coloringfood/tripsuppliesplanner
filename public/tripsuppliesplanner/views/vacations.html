Welcome to the Vacations Page

This page lists all the planned vacations you have (NOTE: currently it shows everyone's vacations until I get the vacation invite system working). It also allows you to create a new vacation to start packing for.

Instructions for making a Vacation
  • Click the Create Vacation Button below
  • Fill in the name of the vacation
  • Select the starting and ending dates for your vacation
  • Select what type of vacation this is, as well as any activities or factors that will apply to it
    • Vacation types:
      • If you are planning on a vacation to span a few activities, select the different types you will hav as well as how many days of the vacation will be each event.
      • ie: If you are planning on taking an RV to the redwoods, then camping for 3 days you may select RV and specify 2 days, and then select camping and specify 3 days
      • Note: the days is optional, if you don't specify the days we will assume it will span the whole vacation
    • Activities:
      • Specify any activities you are planning on doing on this vacation, also specify how many times/days you will be doing that activity
      • Note: the days is optional, if you don't specify the days we will assume it will span the whole vacation
    • Other:
      • Specify any of the other factors that will apply to the vacation. There are no days applied to these, so they will just span the whole vacation
